 Stranger Things Microsite 

 Project Summary
This project is a Vite-powered, single-page microsite built with vanilla HTML/CSS/JavaScript.

Main modules:
1. Landing, About, Cast, AR Experience, Upside Down, Trailer sections.
2. AR face-swap flow using Hugging Face Spaces + Gradio backend.
3. Trailer modal with reliable YouTube audio stop on close.
4. Download and Share actions for generated AR output.

Tech Stack
1. Frontend: HTML5, CSS3, Vanilla JavaScript (ES modules).
2. Build/Dev: Vite.
3. AR API integration: `@gradio/client` loaded via CDN ESM import.
4. Backend host: Hugging Face Spaces (Gradio app).
